## Lessons Learned

- Console filter, Toggle on speech bubble in Console window (top-right corner).
- Unity puts all scripts in the same assembly (two classes named UIController will break the build).
- Always delete Unity assets through the Project window — never through Windows Explorer. Unity tracks every file with a hidden .meta file (it stores the GUID that scenes and prefabs use to find assets). If you delete via Explorer, the .meta orphan stays behind and causes reference errors.
